# 教路狂奔：老师的极速通勤
软件学院软件开发课程作业

## 游戏概述
本游戏《教路狂奔：老师的通勤之路》是饭寒暄粉丝会的训练项目，游戏背景设定为老师往返两个校区之间通勤的场景。

## 快速开始
1. 点击 `Game.exe` 启动游戏。
2. **账号注册/登录**：
   - 新用户需先注册账号，已有账号可直接登录。
   - 账号密码要求：1-10位字符串，由数字、大小写字母组成（中间不含空格）。

## 游戏规则
### 1. 关卡与载具
- 初始解锁：第1关 + 海陆空各1款初始载具。
- 关卡分类：
  - 陆地关卡：1、4、7关
  - 空中关卡：2、5、8关
  - 海上关卡：3、6、9关
- 难度设定：关卡难度随序号依次递增。
- 载具研发：通过关卡收集零件解锁新载具，零件换算规则：
  - 金零件 = 3个普通零件
  - 银零件 = 2个普通零件
  - 铜零件 = 1个普通零件

### 2. 核心玩法
- **油量机制**：
  - 碰撞障碍物会减少油量，拾取油箱可增加油量。
  - 油量为0时游戏失败。
  - 关卡通关条件：行驶里程达到1000单位（老师抵达目的地）。
  - 关卡解锁规则：需通关前一关才能解锁下一关。

#### 陆地关卡（汽车）
- 操作方式：点击「上」按键控制汽车跳跃躲避障碍物。
- 特殊机制：踩在障碍物顶部可触发二次跳跃。

#### 空中关卡（飞机）
- 操作方式：点击「上」按键控制飞机起飞。
- 飞行机制：起飞后飞机会短暂跳跃，随后开始向下坠落。
- 失败条件：飞机驶出屏幕边界则游戏失败。

#### 海上关卡（船只）
- 操作方式：使用「上/下」按键控制船只移动。
- 油量收集：船只可拾取当前航道及上一航道的油箱。

### 3. 暂停功能
随时按下「空格键」可暂停游戏。

## 注意事项
### 1. 游戏启动
- 确保 `Game.exe` 所在目录下的所有游戏资源文件（如贴图、音频、配置文件）保持完整，文件缺失可能导致游戏启动失败或运行异常。
- 直接双击打包文件夹中的 `Game.exe` 即可启动游戏，无需额外安装。

### 2. 运行环境
- 游戏依赖 **Java 运行环境（JRE）1.8** 才能正常运行。
- 打包文件夹中已包含完整的 JRE 1.8 环境，无需在系统中单独安装。

### 3. 兼容性
- 推荐操作系统：Windows 10/11（64位）
- 已知问题：因 JRE 兼容性限制，游戏在 Windows 7 及更低版本系统中可能无法正常运行。
